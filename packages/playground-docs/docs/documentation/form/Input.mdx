`Input` is a component to display input fields with a label. It is basically a wrapper around the native `input` element.

## Example

<Example name="Form/Input" />

<Features features={[
  "Multiple sizes",
  "Multiple states",
]} />

## API

### \{:component} Input

#### Usage

```tsx
<Input
  label="Label"
  placeholder="Placeholder"
  size="medium"
  state="default"
  value={value}
  onChange={onChange}
/>
```

### \{:type} InputSize

The size of the input field.

```ts
type InputSize = 'small' | 'medium' | 'large';
```

### \{:type} InputState

The state of the input field.

```ts
type InputState = 'default' | 'error' | 'success' | 'warning';
```
