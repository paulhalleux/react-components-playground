`Badge` is a component to display a badge. It can be used to display a status, a count, or any other information.

## Example

<Example name="Badge" highlight="22-24,27-29,32-34" />

<Features features={[
  'Multiple variants & sizes',
  'Pill-shaped badges',
  'Closeable badges',
]} />

## API

### Badge Props

<Properties properties={[
  { name: 'children', type: 'React.ReactNode', description: 'The content of the badge.' },
  { name: 'size', type: 'BadgeSize', description: 'The size of the badge.', default: '"medium"' },
  { name: 'variant', type: 'BadgeVariant', description: 'The variant of the badge.', default: '"default"' },
  { name: 'pill', type: 'boolean', description: 'Whether the badge should be pill-shaped.', default: 'false' },
  { name: 'closeable', type: 'boolean', description: 'Whether the badge should be closeable.', default: 'false' },
  { name: 'onClose', type: '() => void', description: 'Callback when the badge is closed.' },
  { name: 'onClick', type: '() => void', description: 'Callback when the badge is clicked.' },
]} />

### BadgeSize type

The `BadgeSize` type is an enum with the following values:

```ts
type BadgeSize = "small" | "medium" | "large"
```

### BadgeVariant type

The `BadgeVariant` type is an enum with the following values:

```ts
type BadgeVariant = "primary" | "secondary" | "default" | "danger" | "success" | "info" | "ghost"
```
