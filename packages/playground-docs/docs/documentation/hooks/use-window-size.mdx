`useWindowSize` is a hook that returns the current window size. It also updates the window size when the window is resized.

## Usage

Simply import the hook and use it in your component. The hook returns an object with the current window size.

```tsx
import { useWindowSize } from '@paulhalleux/react-playground';
const App = () => {
  const { width, height } = useWindowSize();
  return (
    <div>
      <p>Window width: {width}</p>
      <p>Window height: {height}</p>
    </div>
  );
};
```

## Reference

```tsx
function useWindowSize(
  window: Window = window
): { width: number; height: number };
```

### Arguments

- `Window object` _(optional)_ - The window object to use. Defaults to `window`.

### Return value

- `width` - The current window width.
- `height` - The current window height.
