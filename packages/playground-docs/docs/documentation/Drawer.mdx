`Drawer` is a component to display a modal window on the side of the screen.

## Example

<Example name="Drawer" highlight="17-33" />

## API

### \{:component} Drawer

Drawer component is the main component to display a modal window. It is composed of 3 main sub-components: `Drawer.Header`, `Drawer.Body` and `Drawer.Footer`.

#### Usage

```tsx
<Drawer>
  <Drawer.Header>
    <Drawer.Title>Drawer title</Drawer.Title>
  </Drawer.Header>
  <Drawer.Body>
    <p>Drawer body text goes here.</p>
  </Drawer.Body>
  <Drawer.Footer>
    <Button variant="secondary">Close</Button>
    <Button variant="primary">Save changes</Button>
  </Drawer.Footer>
</Drawer>
```

#### Properties

<Properties properties={[
  { name: 'children', type: 'ReactNode', description: 'The content of the modal.' },
  { name: 'className', type: 'string', description: 'Additional class name to apply to the modal.' },
  { name: 'open', type: 'boolean', description: 'Whether the modal is open or not.' },
  { name: 'onClose', type: '() => void', description: 'Callback called when the modal is closed.' },
  { name: 'size', type: 'ModalSize', description: 'The size of the modal.', default: 'medium' },
  { name: 'closeOnBackdropClick', type: 'boolean', description: 'Whether the modal should be closed when the backdrop is clicked.', default: 'true' },
  { name: 'minHeight', type: 'number', description: 'The minimum height of the modal.' },
  { name: 'position', type: 'DrawerPosition', description: 'The position of the drawer modal.', default: 'right' },
]}/>

### \{:component} Drawer.Header

`Drawer.Header` is a component to display a header in a modal window. It is usually used to display a title and a close button.
It has a default padding to all sides but the bottom.

#### Usage

```tsx
<Drawer.Header closeable>Header</Drawer.Header>
```

#### Properties

<Properties properties={[
  { name: 'closeable', type: 'boolean', description: 'Whether the modal is closeable or not.', default: 'false' },
  { name: 'className', type: 'string', description: 'Additional class name to apply to the modal header.' },
]}/>

### \{:component} Drawer.Body

`Drawer.Body` is a component to display a body in a modal window. It is usually used to display the main content of the modal.
By default, the body has padding to all sides.

#### Usage

```tsx
<Drawer.Body>Body</Drawer.Body>
```

#### Properties

<Properties properties={[
  { name: 'className', type: 'string', description: 'Additional class name to apply to the modal body.' },
]}/>

### \{:component} Drawer.Footer

`Drawer.Footer` is a component to display a footer in a modal window. It is usually used to display buttons.
By default, the footer has padding to all sides but the top.

#### Usage

```tsx
<Drawer.Footer>Footer</Drawer.Footer>
```

#### Properties

<Properties properties={[
  { name: 'className', type: 'string', description: 'Additional class name to apply to the modal footer.' },
]}/>

```ts
type DrawerSize = 'small' | 'medium' | 'large' | 'full';
```

### \{:type} DrawerPosition

```ts
type DrawerPosition = 'left' | 'right';
```
