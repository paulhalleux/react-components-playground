### \{:component} Header.Navigation

A simple wrapper around the `Navigation` component. However, is does not allow to define the `size` of the navigation.

#### Usage

```tsx
<Header.Navigation>
  <Navigation.Item>Home</Navigation.Item>
  <Navigation.Item>Products</Navigation.Item>
  <Navigation.Item>Company</Navigation.Item>
</Header.Navigation>
```

#### Properties

<Properties component="Header.Navigation" />

### \{:component} Header.Actions

Will display the children with a `flex` layout and a gap between each element.

#### Usage

```tsx
<Header.Actions>
  <Button>Sign in</Button>
  <Button variant="primary">Sign up</Button>
</Header.Actions>
```

#### Properties

<Properties component="Header.Actions" />

### \{:type} HeaderLayout

`HeaderLayout` is a type that can be used to define the layout of the header.

```ts
type HeaderLayout = "start" | "3-column" | "2-column" | "centered" | "end" | "spaced";
```
