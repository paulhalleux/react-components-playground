`Flex` is a component used to layout content in a page. It is based on the [flexbox](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout/Basic_Concepts_of_Flexbox) layout model.

## Examples

<Example name="components/Layout/Flex/Default" />

## API

### \{:component} Flex

#### Usage

```tsx
<Flex>
  <Flex.Item>Item 1</Flex.Item>
  <Flex.Item>Item 2</Flex.Item>
  <Flex.Item>Item 3</Flex.Item>
</Flex>
```
