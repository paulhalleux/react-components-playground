### \{:type} ButtonSize

The size of the button. Can be one of the following:

```ts
type ButtonSize = 'small' | 'medium' | 'large';
```

### \{:type} ButtonVariant

The variant of the button. Can be one of the following:

```ts
type ButtonVariant = 'default' | 'primary' | 'danger' | 'success' | 'warning';
```
