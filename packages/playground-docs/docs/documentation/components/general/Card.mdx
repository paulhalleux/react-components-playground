`Card` is a component that displays content in a styled container. It can be used to display a variety of content types, such as text, images, and icons.

## API

### \{:component} Card

Main card component.

#### Usage

```tsx
<Card>
  <Card.Header>
    Card Header
  </Card.Header>
  <Card.Body>
    Some quick example text to build on the card title and make up the bulk of the card's content.
  </Card.Body>
  <Card.Footer>
    <Button variant="primary">Go somewhere</Button>
  </Card.Footer>
</Card>
```
