`Progress` is a component that indicates the status of a process. It can be used to show the progress of a file upload, a download, an installation, or any other process that is not instantaneous.

### \{:type} ProgressVariant

The variant of the progress bar.

```ts
type ProgressVariant =
  | "default"
  | "info"
  | "success"
  | "warning"
  | "danger"
  | "primary";
```

### \{:type} ProgressSize

The size of the progress bar.

```ts
export type ProgressSize = "small" | "medium" | "large";
```
