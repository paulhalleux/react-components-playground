An empty state component for when there is no data to display. It can be used to display a message, an icon and actions to the user.
It adapts to the container size.

## Examples

<Example name="components/General/EmptyState" />

## API

### \{:component} EmptyState

#### Usage

```tsx
<EmptyState
  title="Component not found"
  description="The component you are looking for does not exist."
  actions={[{ type: "button", onClick: () => {}, label: "Go back" }]}
  icon={CleanIcon}
/>
```

#### Properties

<Properties component="EmptyState" />

### \{:type} EmptyStateAction

The empty state action type. Either a button or a link.

```ts
type EmptyStateAction =
  | {
      type: "button";
      onClick: () => void;
      label: string;
    }
  | {
      type: "link";
      href: string;
      label: string;
    };
```

### \{:type} EmptyStateVariant

The variant to display the empty state in.

```ts
type EmptyStateVariant = "default" | "error";
```
