`PopConfirm` is a confirm dialog component based on `Popover`. It is used to confirm an action or ask for a confirmation before an action.

It is similar to `ConfirmDialog` but it is based on `Popover` instead of `Modal`. It is more suitable for inline actions.

## API

### \{:component} PopConfirm

#### Usage

```tsx
<PopConfirm
  title="Are you sure to delete this task?"
  description="This action cannot be undone."
  onConfirm={() => console.log('confirmed')}
  onCancel={() => console.log('canceled')}
>
  <Button>Delete</Button>
</PopConfirm>
```
