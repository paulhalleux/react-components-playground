## API

### \{:component} Tabs

`Tabs` is a wrapper component that contains multiple `Tab` components. It handles the state of the tabs and renders the tab list.

#### Usage

```tsx
<Tabs>
  {/** Tabs */}
</Tabs>
```

#### Properties

<Properties component="Tabs" />

### \{:component} Tab

`Tab` is a component that represents a single tab. It is used as a child of the `Tabs` component.

#### Usage

```tsx
<Tab label="Tab 1" id="tab1">
  {/** Tab content */}
</Tab>
```
