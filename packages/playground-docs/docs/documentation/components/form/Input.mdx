`Input` is a component to display input fields with a label. It is basically a wrapper around the native `input` element.

## API

### \{:component} Input

#### Usage

```tsx
<Input
  placeholder="Placeholder"
  size="medium"
  state="default"
  value={value}
  onChange={onChange}
/>
```

#### Properties

Properties are the same as the native `textarea` element, with the following exceptions:

- `size` is of type `InputSize` and represents the size of the input field.
- `state` is of type `InputState` and represents the state of the input field.
- `value` is of type `string` and represents the value of the input field.
- `onChange` is of type `(value: string) => void` and represents the callback when the value of the input field changes.

### \{:type} InputSize

The size of the input field.

```ts
type InputSize = 'small' | 'medium' | 'large';
```

### \{:type} InputState

The state of the input field.

```ts
type InputState = 'default' | 'error' | 'success' | 'warning';
```
