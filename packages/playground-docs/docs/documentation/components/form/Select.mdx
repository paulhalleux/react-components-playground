`Select` is a component that renders a select input. It is a styled version of the `select` HTML element.
For multiple selection, use the `MultipleSelect` component.

## Examples

<Example name="components/Form/Select" />

## API

### \{:component} Select

#### Usage

```tsx
<Select
  value={value}
  onChange={onChange}
  options={[
    { value: '1', label: 'Option 1' },
    { value: '2', label: 'Option 2' },
    { value: '3', label: 'Option 3' },
  ]}
/>
```

#### Properties

<Properties component="Select" />

### \{:type} SelectOption

Represents an option in the select input.

```ts
type SelectOption = {
  value: string;
  label: string;
  disabled?: boolean;
};
```
