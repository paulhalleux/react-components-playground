## API

### \{:component} Table

#### Usage

```tsx
<Table
  columns={columns}
  data={data}
  maxHeight={300}
/>
```

<Properties component="Table" />

### \{:type} TableColumn

`TableColumn` is a type that describes the columns of the table.

```typescript
type TableColumn<T extends Record<string, any>> = {
  key: keyof T;
  label: string;
  render?: (value: T[keyof T]) => React.ReactNode;
  renderHeader?: (label: string) => React.ReactNode;
  sortable?: boolean;
  sortFn?: (a: T, b: T, order: "asc" | "desc" | null) => number;
  sort?: "asc" | "desc" | null;
  width?: string | number;
};
```
