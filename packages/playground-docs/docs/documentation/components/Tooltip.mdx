`Tooltip` is a component that displays a tooltip when the user hovers over it. It is build around the `Popover` component.

## Example

<Example name="Tooltip" />

## API

### \{:component} Tooltip

Wrap the content you want to display in a tooltip with the `Tooltip` component.

#### Usage

```tsx
<Tooltip content="This is a tooltip">
  <Button>Hover me</Button>
</Tooltip>
```

#### Props

<Properties component="Popover" omit={["trigger", "content"]} additional={[
  { name: "content", type: "ReactNode", default: "undefined", description: "The content of the tooltip.", required: true },
]} />

### \{:type} TooltipVariant

The variant of the tooltip.

```ts
type TooltipVariant = 'default' | 'secondary';
```
