`CodeBlock` is a component that displays a block of code.
If the height of the code block exceeds the value of `collapseAt` property, it will be collapsed and a button will be displayed to expand it.

<Example name="CodeBlock" highlight="25" />

## API

### \{:component} CodeBlock

#### Usage

```tsx
<CodeBlock
  language="typescript"
  collapseAt={300}
  defaultExpanded={false}
>
  {`const a = 1;`}
</CodeBlock>
```
