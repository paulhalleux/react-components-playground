`Loader` is a component to display placeholders while loading data.

<Example name="Loader" highlight="12" />

## API

### \{:component} Loader

#### Usage

```tsx
<Loader size="medium" orientation="horizontal" color="primary" label="Loading..." />
```

#### Properties

<Properties component="Loader" />

### \{:type} LoaderSize

The size of the loader. The default value is `medium`.

```ts
type LoaderSize = 'small' | 'medium' | 'large';
```

### \{:type} LoaderOrientation

The orientation of the loader. The default value is `horizontal`.

If the orientation is `horizontal`, the loader label will be displayed on the right of the loader. Otherwise, the loader label will be displayed below the loader.

```ts
type LoaderOrientation = 'horizontal' | 'vertical';
```
