`Skeleton` is a component to display placeholders while loading data. It can be used to display a placeholder for a list of items, or for a single item.

## Example

<Example name="Skeleton" highlight="7-23" />

## API

### \{:component} Skeleton

`Skeleton` is a component to display a single skeleton loader. It can be configured with a specific width, height, margin and border radius (CSS properties).

#### Usage

```tsx
<Skeleton width={100} height={100} margin={10} />
```

#### Properties

<Properties properties={[
  { name: "width", type: "number | string", description: "Width of the skeleton.", default: "100%" },
  { name: "height", type: "number | string", description: "Height of the skeleton.", default: "100%" },
  { name: "margin", type: "number | string", description: "Margin of the skeleton.", default: "0" },
  { name: "radius", type: "number | string", description: "Border radius of the skeleton.", default: "3px" },
]} />

### \{:component} Skeleton.Container

`Skeleton.Container` is a component to display a list of skeletons. It can be configured to display the skeletons in a row or in a column, and to align them.

#### Usage

```tsx
<Skeleton.Container>
  <Skeleton />
  <Skeleton />
  <Skeleton />
</Skeleton.Container>
```

#### Properties

<Properties properties={[
  { name: "direction", type: '"row" | "column"', description: "Skeleton direction.", default: '"row"' },
  { name: "alignItems", type: '"flex-start" | "center" | "flex-end"', description: "Alignment of skeleton items.", default: '"flex-start"' },
  { name: "justifyContent", type: '"flex-start" | "center" | "flex-end"', description: "Alignment of skeleton items.", default: '"flex-start"' },
  { name: "gap", type: "number | string", description: "Gap between skeleton items.", default: "8px" },
]} />
