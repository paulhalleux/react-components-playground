`Breadcrumb` is a secondary navigation scheme that reveals the user's location in a website or web application.

## Examples

<Example name="components/Navigation/Breadcrumb" />

## API

### \{:component} Breadcrumb

#### Usage

```tsx
<Breadcrumb
  items={[
    { label: 'Home', href: '/' },
    { label: 'Products', href: '/products' },
    { label: 'Product 1', href: '/products/1' },
  ]}
/>
```
