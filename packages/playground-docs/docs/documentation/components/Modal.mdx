## API

### \{:component} Modal

Modal component is the main component to display a modal window. It is composed of 3 main sub-components: `Modal.Header`, `Modal.Body` and `Modal.Footer`.

#### Usage

```tsx
<Modal>
  <Modal.Header>
    <Modal.Title>Modal title</Modal.Title>
  </Modal.Header>
  <Modal.Body>
    <p>Modal body text goes here.</p>
  </Modal.Body>
  <Modal.Footer>
    <Button variant="secondary">Close</Button>
    <Button variant="primary">Save changes</Button>
  </Modal.Footer>
</Modal>
```

<Properties component="Modal" />

### \{:component} Modal.Header

`Modal.Header` is a component to display a header in a modal window. It is usually used to display a title and a close button.
It has a default padding to all sides but the bottom.

#### Usage

```tsx
<Modal.Header closeable>Header</Modal.Header>
```

#### Properties

<Properties component="ModalHeader" />


### \{:component} Modal.Body

`Modal.Body` is a component to display a body in a modal window. It is usually used to display the main content of the modal.
By default, the body has padding to all sides.

#### Usage

```tsx
<Modal.Body>Body</Modal.Body>
```

#### Properties

<Properties component="ModalBody" />

### \{:component} Modal.Footer

`Modal.Footer` is a component to display a footer in a modal window. It is usually used to display buttons.
By default, the footer has padding to all sides but the top.

#### Usage

```tsx
<Modal.Footer>Footer</Modal.Footer>
```

### \{:type} ModalSize

`ModalSize` is a type to define the size of a modal.

```ts
type ModalSize = 'small' | 'medium' | 'large' | 'full';
```
