`Toast` is a component that displays a message to the user. It is used to provide feedback about an operation or inform something important.

## API

### ToasterProvider

The `ToasterProvider` component is used to display the `Toast` components. It should be placed at the root of your application.

<Properties properties={[
  { name: "replace", type: "boolean", description: "Whether new toast should replace previous.", default: "false" },
  { name: "position", type: 'ToasterPosition', description: "The position of the toasts.", default: '"top-right"' },
]} />

```tsx
import { ToasterProvider } from "@paulhalleux/react-playground";

const App = () => (
  <ToasterProvider>
    <App />
  </ToasterProvider>
);
```

### ToasterPosition

```ts
type ToasterPosition = "top-left" | "top-right" | "bottom-left" | "bottom-right";
```

### useToaster

The `useToasts` hook returns a `pushToast` function that can be used to push a new toast to the `ToasterProvider`.

```ts
import { useToasts } from "@paulhalleux/react-playground";

const { pushToast } = useToaster();

// --> pushToast(toast: ToastType) => void
```

### Toast Type

```ts
type Toast = {
  icon?: React.FC<IconProps>;
  title?: string;
  content: string;
  type?: "primary" | "success" | "warning" | "error" | "default";
  duration?: number;
  closable?: boolean;
  actionLabel?: string;
  onAction?: () => void;
};
```
