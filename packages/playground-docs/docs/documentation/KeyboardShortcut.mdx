`KeyboardShortcut` is a component that displays a keyboard shortcut. It can also handle the keyboard shortcut when defining a `onShortcut` callback.

]} />

## API

### \{:component} KeyboardProvider

`KeyboardProvider` is a component that handles keyboard shortcuts. It registers a `keydown` event listener on the `window` object.

It should wrap all the `KeyboardShortcut` components to handle the keyboard shortcuts.

```tsx
<KeyboardProvider>
  {/* ... */}
</KeyboardProvider>
```

### \{:component} KeyboardShortcut

#### Usage

```tsx
<KeyboardShortcut shortcut={{
  ctrlKey: true,
  key: "s",
}} />
```

#### Properties

<Properties properties={[

```ts
type Shortcut = {
  ctrlKey?: boolean;
  altKey?: boolean;
  shiftKey?: boolean;
  metaKey?: boolean;
  key: string;
```
