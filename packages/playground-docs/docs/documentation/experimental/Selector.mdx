`Selector` is a component that allows you to select elements in a container.
It works like a lasso tool, where you click and drag to select elements.

<Example name="Selector" />

## API

### \{:component} Selector

#### Usage

```tsx
<Selector parentRef={parentRef} color={[255, 0, 0]}>
  {/** Selectable items */}
</Selector>
```

#### Properties

<Properties component="Selector" />

### \{:function} selectable

The `selectable` function is a HOC that allows you to make an element selectable. It will register itself to the `Selector` component.

```tsx
export const SelectableItem = selectable((props) => {
  return <div {...props} />;
});
```
